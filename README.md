🧠 What is Research Desk OS?

Research Desk는 브라우저에서 작동하는 로컬 기반 리서치 시스템입니다.
단순 메모가 아니라 가설(Hypothesis) 중심으로 기록하고,
알람/링크/리포트로 구조화하여 판단 히스토리를 쌓습니다.

✔ 서버 없음 · ✔ 로그인 없음
✔ 로컬 저장 · ✔ 즉시 실행

📌 Core Features
✍️ Desk (분석 데스크)

종목명/코드 입력

핵심 가설 기록 + URL 자동 미리보기

무효화 조건(Risk) 입력

태그/섹터 분류

매매 기록 옵션

이미지 캡처(3장) + 메모/펜 메모

가설 ↔ 가설 연결(Link)

댓글

🔍 Explorer (탐색 & 지식 맵)

글로벌 검색 (가설/텍스트/종목/태그)

알람 대기 가설

리뷰 대상 가설 (시간 경과)

지식 네트워크(Link Map)

🧾 Report (리포트)

대표 종목/시세 입력

근거 가설 카드 포함

Executive / Macro / Sector / Conclusion 섹션

섹션별 카드 연결

PDF/Print 지원

📊 Sync Monitor (상단 상태 표시)

Local DB 최신 시각

Import / Export 시각

전체 시세 동기화 시각

❗ 경고/위험 상태는 데이터 백업 여부를 함께 보여줍니다.
