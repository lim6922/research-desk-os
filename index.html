📘 Research Desk v10.5 가이드

🧠 Research Desk 개요
Research Desk는 단순 메모장이 아니라,
가설 → 근거 → 무효화 조건 → 검증 구조를 갖춘 리서치 시스템입니다.
모든 기록은 시간순으로 축적되며 판단의 히스토리가 됩니다.

📌 핵심 원칙
1) 반드시 가설 + 근거(URL)를 함께 기록합니다.
2) 반드시 무효화 조건(Risk)을 명시합니다.
3) 기록은 검증을 위한 자산입니다.

🧭 기본 사용 흐름

① 종목명 + 코드 입력
② Ctrl + Shift + C → 종목 코드 조회
③ 가설 + 근거 + 리스크 입력
④ (선택) 매매 기록 활성화
⑤ Ctrl + Enter → 저장

이미지는 Ctrl + V로 붙여넣기 가능 (최대 3장).
클릭 시 확대 + 펜 메모 저장 가능.

⌨️ 단축키 안내

📌 전역
• Ctrl + Enter : 저장 (Desk / Report)
• Ctrl + Shift + F : 현재 탭 검색창 이동
• Ctrl + Shift + C : 종목 코드 조회
• Ctrl + Shift + Q : 현재 종목 시세 갱신
• Ctrl + Alt + Q : 전체 시세 동기화
• Ctrl + Alt + D : Desk 탭 이동
• Ctrl + Alt + M : Monitor 탭 이동
• Ctrl + Alt + E : Explorer 탭 이동
• Ctrl + Alt + R : Report 탭 이동

🧩 Markdown 에디터
• Desk의 "핵심 가설" 입력창 하단 버튼으로 제목/링크/코드/표/체크리스트 템플릿을 빠르게 삽입할 수 있습니다.
• 지원 렌더링: 제목, 굵게/기울임, 인용, 구분선, 목록(순서/비순서), 체크리스트, 링크, 인라인 코드
